HTML
HTMLやXMLはマークアップ言語で呼ばれ、プログラミング言語のように制御構文やコレクションクラスは持たない。タグで作成するスタイルをとる。
CSS
Cascading Style Sheetの略。HTMLがWebページに構造を与えるのに対し、CSSはその構造の色やフォント、配置などを与えるために使用する。CSSのセレクタはJavaScriptでも使用可能である。
JavaScript
HTMLやCSSをコントロールできる。もともとはブラウザベースで動作するように設計されたが、通常のプログラミング言語と同様に制御構文、コレクションクラスを持つのでサーバサイドとしても使用できる。サーバサイドというとNode.jsを連想する人も多いが、10数年前から既にサーバサイドで使用されていた。
